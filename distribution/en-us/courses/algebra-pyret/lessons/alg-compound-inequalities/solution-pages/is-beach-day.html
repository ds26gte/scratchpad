Word Problem: is-beach-day

Referenced from lesson Compound Inequalities

Directions: Use the Design Recipe to write a function is-beach-day, which takes in a temperature in Fahrenheit and a forecast and determines if it is beach weather: hot and sunny

Contract and Purpose Statement

Every contract has three parts…​

# is-beach-day::Number, String->Boolean

# Consumes a temperature and forecast and returns true if it is both hot and sunny

Examples

Write some examples, then circle and label what changes…​

examples:

__is-beach-day (50, "sunny")is is-hot(50) and is-sunny("sunny")

__is-beach-day (96, "cloudy")is is-hot(96) and is-sunny("cloudy")

__is-beach-day (84, "sunny")is is-hot(84) and is-sunny("sunny")

end

Definition

Write the definition, giving variable names to all your input values…​

fun is-beach-day(temp, forecast):

__is-hot(temp) and is-sunny(forecast)

end
