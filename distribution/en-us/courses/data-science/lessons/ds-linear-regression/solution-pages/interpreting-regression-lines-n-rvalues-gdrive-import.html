Interpreting Regression Lines & r-Values

Referenced from lesson Linear Regression

Each description on the left is written about the linear regression findings on the right. Fill in the blanks using the information in the line of best fit and the r-value.

 

1

For every additional Marvel Universe movie released each year, the average person is predicted to consume 3.19[amount] fewer[more / fewer] pounds of sugar! This correlation is non-existent[strong, moderate, weak, non-existent].

 
 y = −3.19x + 12
 r = −0.05

2

Shoe size and height are strongly[strongly, moderately, weakly, not], positively[positively / negatively] correlated. If person A is one size bigger than person B, we predict that they will be roughly 1.65[amount] inches taller than person B as well.

 
y = y=1.65x + 52
r = 0.89

3

There is no[a strong, a moderate, no] relationship found between the number of Uber drivers in a city and the number of babies born each year.

 
y = 0.012x + 7.8
r = 0.01

4

The correlation between weeks-of-school-missed and SAT score is moderate[strong, moderate, weak, non-existent] and negative[positive / negative]. For every week a student misses, we predict a 15.3[amount] point drop[gain / drop] in their SAT score.

 
y = -15.3x + 1150
r = −0.65

5

There is a weak[strong, moderate, weak, non-existent], positive[positive / negative] correlation between the number of streaming video services someone has, and how much they weigh. For each service, we expect them to be roughly 1.6[amount] pounds heavier.

 
y = 1.6x + 160
r = 0.12
