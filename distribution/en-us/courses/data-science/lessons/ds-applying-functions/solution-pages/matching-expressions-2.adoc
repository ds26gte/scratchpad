[cols=">.^15a,^.^1a,2,^.^1a,.^10a",stripes="none",grid="none",frame="none", options="header"]
|===

| is-capital +::+ (city +::+ String, state +::+ String) -> Boolean
| *1*  +(C)+ || *A* |`show-score("Vikings", 41)`
@vspace{1ex}

| is-capital +::+ (city +::+ String, pop +::+ Number) -> Boolean
